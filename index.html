MCP to Skill

Turn any MCP server into an OpenClaw or Hermes skill

Copy the command below. Run it in your terminal.

Notion 
npx @filiksyos/mcptoskill https://mcp.notion.com/mcp
PostHog 
npx @filiksyos/mcptoskill https://mcp.posthog.com/mcp
Supabase 
npx @filiksyos/mcptoskill https://mcp.supabase.com/mcp
Context7 
npx @filiksyos/mcptoskill https://mcp.context7.com/mcp
Exa MCP 
npx @filiksyos/mcptoskill https://mcp.exa.ai/mcp --name=exa
Render 
npx @filiksyos/mcptoskill https://mcp.render.com/mcp --header "Authorization: Bearer YOUR_RENDER_API_KEY"

Get API key: Account Settings → API Keys

+ Custom MCP 
npx @filiksyos/mcptoskill <mcp-url>

Replace <mcp-url> with your MCP server URL

Run this in your terminal
